API Documentation

Interface REST pour la gestion des abonnements

Version 1.0
Base URL
http://localhost:4567
Tous les endpoints sont préfixés par cette URL de base.
GET
Test de connexion
/api/abonnements

Description : Récupère la liste complète des abonnements

Réponse (200 OK) :
[ { "id": "123e4567-e89b-12d3-a456-426614174000", "clientName": "Jean Dupont", "nomService": "Netflix", "dateDebut": "2024-01-01", "dateFin": "2024-12-31", "categorie": "Streaming", "prixMensuel": 15.99, "actif": true, "derniereUtilisation": "2024-11-15" } ]
GET
Récupérer un abonnement
/api/abonnements/{id}

Description : Récupère un abonnement spécifique par son ID

Paramètres :
  • id (UUID) : Identifiant unique de l'abonnement
POST
Créer un abonnement
/api/abonnements

Description : Crée un nouvel abonnement

Corps de la requête (JSON) :
{ "clientName": "Marie Martin", "nomService": "Spotify Premium", "dateDebut": "2024-01-01", "dateFin": "2024-12-31", "categorie": "Musique", "prixMensuel": 9.99 }
PUT
Modifier un abonnement
/api/abonnements/{id}

Description : Met à jour un abonnement existant

DELETE
Supprimer un abonnement
/api/abonnements/{id}

Description : Supprime un abonnement par son ID

POST
Import JSON
/api/abonnements/import

Description : Importe une liste d'abonnements depuis un JSON

Corps de la requête :
Content-Type: application/json [ { "clientName": "Client 1", "nomService": "Service 1", "dateDebut": "2024-01-01", "dateFin": "2024-12-31", "categorie": "Category", "prixMensuel": 10.00 } ]
Codes de réponse HTTP
  • 200 OK Requête réussie
  • 201 Created Ressource créée avec succès
  • 404 Not Found Ressource non trouvée
  • 400 Bad Request Données invalides
  • 500 Internal Server Error Erreur serveur
Exemples d'utilisation
JavaScript (fetch) :
// Récupérer tous les abonnements const response = await fetch('/api/abonnements'); const abonnements = await response.json(); // Créer un nouvel abonnement const newAbonnement = { clientName: 'John Doe', nomService: 'Netflix', dateDebut: '2024-01-01', dateFin: '2024-12-31', categorie: 'Streaming', prixMensuel: 15.99 }; const response = await fetch('/api/abonnements', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(newAbonnement) });
cURL :
# GET - Liste des abonnements curl -X GET http://localhost:4567/api/abonnements # POST - Créer un abonnement curl -X POST http://localhost:4567/api/abonnements \ -H "Content-Type: application/json" \ -d '{ "clientName": "Test User", "nomService": "Test Service", "dateDebut": "2024-01-01", "dateFin": "2024-12-31", "categorie": "Test", "prixMensuel": 10.00 }'
