## Accès au QR Code

Le QR code est disponible sous 2 formats:

### 1️⃣ **Page HTML Interactive** 
📄 Fichier: `qr-code.html`
- Ouvre automatiquement le QR code dans le navigateur
- Bouton "Télécharger QR Code" en PNG
- Lien direct vers le repo GitHub
- Format: **HTML + JavaScript (QRCode.js)**

**Pour accéder:**
```bash
# Ouvre dans le navigateur
http://localhost:4567/qr-code.html
```

### 2️⃣ **URL AWS QR Code Generator**
Lien direct pour générer l'image PNG:
```
https://api.qrserver.com/v1/create-qr-code/?size=500x500&data=https://github.com/AzizTliliMiageM1/Projet-Dev-Ops.git
```

**Pour inclure dans PowerPoint:**
- Clic droit → "Enregistrer l'image sous"
- Glissez-déposez directement dans la diapositive

## 📊 Spécifications du QR Code

| Paramètre | Valeur |
|-----------|--------|
| **Format** | PNG |
| **Taille** | 500x500px |
| **Couleur fond** | Blanc |
| **Couleur QR** | Bleu (#667eea) |
| **Niveau correction** | H (30% de perte) |
| **Contenu** | URL GitHub repo |

## 🎯 Comment utiliser en présentation?

### Option 1: Afficher la page HTML
```md
1. Ouvre qr-code.html dans le navigateur
2. Projette l'écran pendant la présentation
3. Les auditeurs scannent avec leur téléphone
```

### Option 2: Inclure l'image dans PowerPoint
```md
1. Télécharge le PNG depuis la page HTML
2. Insère l'image dans la diapositive de fin
3. Ajoute le texte: "Scannez pour accéder au code source"
```

### Option 3: Utiliser l'URL directe
```md
1. Copie l'URL API QRServer
2. Crée une image depuis cette URL dans PowerPoint
3. Configure un lien hypertexte sur l'image
```

## ✅ Avantages du QR Code

- ✅ **Facile d'accès** - Un seul scan pour le repo
- ✅ **Impression possible** - Pour les supports papier
- ✅ **Mobile-friendly** - Fonctionne sur tous les téléphones
- ✅ **Professionnel** - Impression de qualité
- ✅ **Traçable** - Vous pouvez voir les scans (analytics optionnel)
